# 💬 Gup-Sup - Full Stack Real-Time Chat Application

Gup-Sup is a modern full-stack real-time chat application built using the MERN stack. It enables users to communicate instantly through one-to-one messaging with secure authentication, media sharing, and live communication powered by Socket.IO.

> **Project Type:** Collaborative Project (2 Developers)  
> **My Primary Role:** Backend Developer
